Android Chess Project completed in Software Methodology(CS213) at Rutgers University by Shanmukh Koka and Marc Fortich.

Two player chess game with undo and replay functionality. Detects check/checkmate, pawn promotion, castling, enpassant. Allows user to save and replay game at game completion. 

Completed on Android Studio, using Nexus 4 emulator with Android API level 29 and Java 8.

Run MainPageActivity.java to start application.
